# Advanced Contact Form with Live Validation

Цей проєкт є частиною Практичної роботи №6 (Частина 1) з курсу "Основи програмування мовою JavaScript". Це інтерактивна контактна форма з валідацією в реальному часі та сучасним користувацьким інтерфейсом.

## 🌟 Ключові особливості

* **Інтерактивна валідація:** Перевірка даних (ім'я, email, телефон, довжина повідомлення) одразу під час введення.
* **Оптимізація (Debounce):** Скрипт очікує паузу у введенні користувача перед перевіркою, що покращує продуктивність.
* **Візуальний прогрес:** Динамічний прогрес-бар, що показує відсоток успішно заповнених полів форми.
* **Контроль довжини повідомлення:** Вбудований лічильник символів (від 20 до 500) для текстового поля.
* **Розумна кнопка відправки:** Кнопка автоматично розблоковується лише тоді, коли всі дані введено коректно.
* **Сучасний UX:** Плавні CSS-переходи, стани завантаження та чіткі повідомлення про успішне завершення операції.

## 🛠 Технічний стек

* **HTML5:** Використання семантичних тегів та атрибутів доступності (Accessibility).
* **CSS3:** Адаптивний дизайн, використання змінних та анімацій.
* **JavaScript:** Ванільний JS (без бібліотек), застосування делегування подій (Event Delegation) та регулярних виразів.

## 📝 Інструкція з використання

1.  Відкрийте файл `index.html` у будь-якому сучасному веб-браузері.
2.  Заповніть усі поля форми. Слідкуйте за індикаторами (червоний/зелений колір) та прогрес-баром.
3.  Після повного заповнення натисніть кнопку "Відправити".
